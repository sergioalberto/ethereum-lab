# Supply Chain Web3 Client
This is the Web3 client that will use the already smart contract deployed (SupplyChain).

_Note:_ You should create a `.secret` file with your 12 secret words from MetaMask wallet.

## Set up
```shell
npm i
```

## Run
```shell
npm run start
```
